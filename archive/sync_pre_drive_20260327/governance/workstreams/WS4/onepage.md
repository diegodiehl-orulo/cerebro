# WS4 — One-Pager

## O que é
Infraestrutura comercial: CRM, hygiene, handoff e cadências SDR.

## Problema que resolve
CRM sujo → pipeline invisível → decisões erradas.
Handoff indefinido → deals perdidos.
Cadências inexistentes → energia desperdiçada.

---

## Escopo

| Entra | Não entra |
|-------|-----------|
| Hygiene quincenal | Jornada DL (WS2) |
| Handoff BDR→Closer | Ativação territorial (WS3) |
| Cadências SDR | Produtos financeiros (WS7) |
| Evidência quincenal | BI, scoring, RevOps |

---

## 3 Prioridades de H1

1. **Higiene quincenal** — deals com dono + próxima ação + data
2. **Handoff documentado** — critério claro de passagem
3. **Cadências SDR** — evento + DL operando

---

## Indicadores mínimos

- Deals com próximo passo
- Deals sem dono
- Reuniões agendadas (evento + DL)

---

## Fronteira com WS2

- **WS2:** Define a jornada DL — o que acontece com cada cliente
- **WS4:** Garante que o CRM suporte essa jornada — infraestrutura

---

## O que é avanço real

| Avanço | Não é |
|--------|-------|
| Deals atualizados | Discussão |
| Handoff escrito | Intenção |
| Cadência definida | Reunião sem output |
| Baseline registrado | Promessa |
