# 05 — DASHBOARD DE KPIS E CHECK-IN

## O Painel de Controle para Medir o Progresso

**Propósito:** Este documento fornece a estrutura para seu Dashboard de KPIs (Key Performance Indicators) e o processo do seu Ritual de Check-in Semanal. É a ferramenta que transforma o progresso de uma sensação em um fato observável.

---

### PARTE 2: O RITUAL DE CHECK-IN SEMANAL

**Quando:** Toda sexta-feira, 30 minutos.
**Onde:** Na página do seu Dashboard no Notion.

**O Processo:**

1.  **Atualizar os KPIs (10 min):** Preencha os números da semana no seu Dashboard. Seja brutalmente honesto consigo mesmo.

2.  **Revisar o Progresso dos KRs (10 min):** Para cada KR do trimestre (do documento `04_OKRS_METAS_2026`), atualize a confiança de atingimento (de 0 a 100%) e o status:
    *   **Verde (70-100%):** No caminho certo.
    *   **Amarelo (30-69%):** Em risco. Precisa de atenção.
    *   **Vermelho (0-29%):** Fora do caminho. Precisa de uma intervenção séria.

3.  **Refletir e Planejar (10 min):** Responda a 3 perguntas no seu diário ou no Notion:
    *   **O que funcionou esta semana?** (Celebre as vitórias, por menores que sejam).
    *   **O que foi um desafio?** (Identifique os gargalos sem julgamento).
    *   **Qual é a minha principal prioridade para a próxima semana para mover os KRs amarelos/vermelhos para o verde?**

---

### COMO USAR ESTE DOCUMENTO

-   **Crie no Notion:** Use este modelo para construir seu dashboard real no Notion. As barras de progresso e os números podem ser automatizados.
-   **Agende no Calendário:** Bloqueie 30 minutos na sua agenda toda sexta-feira. Este é um compromisso inegociável com sua clareza e seu progresso.
-   **Seja Honesto:** O dashboard so funciona se os dados forem reais. Nao tenha medo de colocar um KPI no vermelho. O vermelho nao e um fracasso, e um sinal para ajustar a rota.
