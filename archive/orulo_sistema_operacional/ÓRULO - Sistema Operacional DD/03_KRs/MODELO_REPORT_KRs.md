**Fonte:** DD - Definição de Estratégia e OKRs 2026 - Diretoria.xlsx (aba "Modelos Reports")
**Copiado para:** 03_KRs em 26/03/2026
**Uso:** Template oficial para check-ins mensais e revisões trimestrais de OKRs

---

## COMO USAR

Este documento contém os dois modelos de report do sistema de OKRs da Órulo:

1. **Check-in Mensal** — para acompanhamento mensal de cada objetivo
2. **Revisão Trimestral** — para revisão profunda a cada trimestre

Ao rodar um report, copie o template correspondente, preencha com os dados reais e salve em `06_REPORTS/` com o nome `REPORT_[MÊS/TRIMESTRE]_[ANO].md`.

## REGRA DE USO

- Report mensal → aba `06_REPORTS/`, arquivo nomeado como `REPORT_[MÊS]_[ANO].md`
- Revisão trimestral → aba `06_REPORTS/`, arquivo nomeado como `REPORT_Q[N]_[ANO].md`
- Após cada report, atualizar os arquivos KR1.md ... KR7.md em `03_KRs/` com o status mais recente
- A planilha `DD - Definição de Estratégia e OKRs 2026 - Diretoria.xlsx` continua como fonte primária de dados numéricos
